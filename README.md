## Featured Project

### [P2P Anomaly Detection with RAG-Powered Explanations](https://github.com/Carolin16/p2p-anomaly-detection)
> Python · ChromaDB · GPT-4o · FastAPI · Streamlit · Docker · Hugging Face Spaces

Detects fraudulent vendor invoices in a Procure-to-Pay cycle using a **Random Forest classifier** and explains every flag with a policy-grounded audit report via a **RAG pipeline** (ChromaDB + GPT-4o).

- 4 anomaly types: overbilling, duplicate invoices, phantom delivery, new vendor risk
- Hybrid RAG: separate ChromaDB collections for past fraud cases and policy/contract docs
- Deployed on Hugging Face Spaces · [Live Demo](https://carolinjames-p2p-anomaly-detection-ui.hf.space)
